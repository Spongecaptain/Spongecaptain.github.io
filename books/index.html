Books and Papers I Read

读书原本是一件私事，仅仅分享读了什么书，可能意义不大。但是通过一个人阅读的书籍，确实能够方便快速地了解他拥有的技术栈。除此之外，对于分享好书，我乐此不疲。

这里仅仅分享个人对一本书的看法，但是人与人的口味不尽相同，因此如果你有不同的意见，请见谅。

Books

以下是 2020 的书籍阅读。

  • 《Java网络编程(第4版)》：比较基础，主要针对于 OIO 式的 Java Socket 编程；

  • 《Pro Git》：非常好的一本书，让你能够掌握 Git 指令的方方面面；

  • 《Java 编程思想（中文第四版）》：事实上我看到 15 章左右便没有看完，比较晦涩，但是讲述了很多 Java 语言设计的核心思路；

  • 《Jeff Friesen (auth.) - Java I_O, NIO and NIO.2 (2015, Apress)》：用于学习 Java IO API 是非常好的选择；

  • 《Maven 实战》：用于理解基于 Meven 的 Java 项目依赖、多模块、打包、测试、部署；

  • 《Tomcat与Java Web开发技术详解（第3版）》：孙卫琴老师写得还是浅显易懂，非常适合初学者掌握 Tomcat 容器；

  • 《MyBatis 从入门到精通》：主要是掌握 MyBatis 的 API，一个简单的 MyBatis+ Spring Boot 的个人项目地址为：https://github.com/Spongecaptain/mybatisDemo

  • 《MySQL 必知必会》：主要学习 MySQL 的脚本，意义不大，直接选择学习菜鸟教程即可；

  • 《MySQL 技术内幕》：用于理解 MySQL 执行引擎的内部机制；

  • 《高性能mysql第三版》：这本书翻译水平一般，不过内容非常好，也是讲解 MySQL 核心知识；

  • 《从 Paxos 到 Zookeeper》：学习这本书之前，最好先看 Google 的 3 篇 paper：mapreduce、GFS、bigtable，这样容易理解 ZooKeeper 推出的目的；

  • 《剑指offer_第二版》：为 ByteDance 日常实习准备的算法书，仅仅刷了一遍；

  • 《Netty 实战》：这并不是一本好书，因为翻译实在粗制滥造，不过可以参考其目录进行 Netty API 的学习；

  • 《readis 开发与运维》：作为 Readis 的入门书籍非常好；

  • Spring 系列丛书：

    • 《Spring 揭秘》
    • 《Spring 实战（第4版）》
    • 《Spring Boot 实战》

    注：Spring 书籍大同小异，我是先学习 API，然后做了一个 Spring Web 小项目，最后才看 Spring 源码。

  • 《数据密集型应用系统设计》：这本书总结性的内容非常好，但是建议在广泛学习相关领域后再看，否则很难掌握；

  • 《调试九法-软硬件错误的排查之道》：这是一本计算机哲学书，主要说明了为啥手册如此重要，为啥面试造火箭，工作拧螺丝；
